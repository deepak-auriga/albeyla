Responsibilities:
- Spin up on incident creation
- Multi-step investigation:
  1. Fetch related logs (time window)
  2. Query recent deployments
  3. Analyze metric correlations
  4. Check infrastructure changes
  5. Search knowledge graph for similar incidents
  6. Generate RCA report with confidence scores
  7. Suggest fixes (rollback, config change, etc.)

Autonomy:
- Per-incident agent instance
- Parallel investigation threads
- Self-terminating on resolution
